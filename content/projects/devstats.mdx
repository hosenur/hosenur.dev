Devstats is a beautifully crafted, highly customizable and fully responsive dashboard for developers, which aggregates stats and activities from different platforms and displays them at one place.

Devstats currently has support for the following platforms:
- GitHub
- WakaTime
- DEV.to
- Hashnode

## Tech Stack
- NextJS
- TailwindCSS
- DaisyUI
